1. Towards a Production-Ready Design

Now that we‚Äôve broken down responsibilities in our Employee Management System using SRP, let’s take the final steps to make the code more robust and extensible:

  • Replace string-based type with a Java Enum
  • Replace System.err with structured logging
  • Introduce an interface for EmployeeSaver to prepare for future extensibility

  • More extensible and testable (via abstractions)

2. Code Enhancements for SRP

‚úÖ 1. Enum for Employee Type

public enum EmployeeType {
    FULLTIME,
    CONTRACT,
    INTERN

‚úÖ 2. Updated Employee Class

public class Employee {

    private final EmployeeType type;

        this.type = type;

‚úÖ 3. Formatter Class (No Change Required)

public class EmployeeFormatter {
    public String formatForHR(Employee emp) {
        return emp.getName() + " - " + emp.getType().name();
    }
}

‚úÖ 4. Create an Interface for Saver

public interface EmployeeSaver {
    void saveToFile(String formattedData) throws IOException;
}

‚úÖ 5. Implement the Interface

import java.io.FileWriter;
import java.io.IOException;

public class EmployeeFileSaver implements EmployeeSaver {
    private final String filePath;

    public EmployeeFileSaver(String filePath) {
        this.filePath = filePath;
    }

    @Override
    public void saveToFile(String formattedData) throws IOException {
        try (FileWriter writer = new FileWriter(filePath, true)) {
            writer.write(formattedData + "\n");
        }
    }
}

‚úÖ 6. Use Structured Logging

import java.util.logging.Level;
import java.util.logging.Logger;

public class EmployeeService {
    private static final Logger logger = Logger.getLogger(EmployeeService.class.getName());

    private final EmployeeFormatter formatter;
    private final EmployeeSaver saver;

    public EmployeeService(EmployeeFormatter formatter, EmployeeSaver saver) {
        this.formatter = formatter;
        this.saver = saver;
    }

    public void save(Employee emp) {
        String formatted = formatter.formatForHR(emp);
        try {
            saver.saveToFile(formatted);
            logger.info("Saved employee: " + formatted);
        } catch (IOException e) {
            logger.log(Level.SEVERE, "Error saving employee data", e);
        }
    }
}

‚úÖ 7. Updated Main Client

public class MainClient {
    public static void main(String[] args) {
        EmployeeFormatter formatter = new EmployeeFormatter();
        EmployeeSaver saver = new EmployeeFileSaver("employee.txt");
        EmployeeService service = new EmployeeService(formatter, saver);

        Employee emp1 = new Employee("Shubham", EmployeeType.FULLTIME);
        Employee emp2 = new Employee("Ashwarya", EmployeeType.CONTRACT);
        Employee emp3 = new Employee("Arvind", EmployeeType.INTERN);

        service.save(emp1);
        service.save(emp2);
        service.save(emp3);
    }
}
